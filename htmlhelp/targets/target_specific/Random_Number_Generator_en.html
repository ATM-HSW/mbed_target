13.2.4. Random Number Generator

This block generates a random 32 bit integer value.

13.2.4.1. Library

Targets/STM/DISCO_F407VG

13.2.4.2. Description

This block generates random 32 bit integer values using the hardware random number generator (RNG) of an STM32 MCU. The generated numbers are true random numbers generated by an analog noise source. For more information about the generation circuit use the reference manual of the used STM32 MCU.

13.2.4.3. Data Type Support

The block outputs an unsigned 32 bit integer (uint32).

13.2.4.4. Parameters and Dialog Box

This block has no parameters to be configured.
