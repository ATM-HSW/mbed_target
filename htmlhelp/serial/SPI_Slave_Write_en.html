5.4.5. SPI Slave Write

This block puts one byte in the SPI output buffer.

5.4.5.1. Library

/SPI

5.4.5.2. Description

This block puts one byte into the output buffer of the SPI peripheral each time it is executed. Since the buffer is only one byte in depth the master is responsible to get the value right in time. If the master doesn’t get the actual value the new value is not put inside the buffer.
