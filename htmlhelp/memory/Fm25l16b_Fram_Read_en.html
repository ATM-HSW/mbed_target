12.1. FM25L16B FRAM Read

Reads one byte stored in a F25L16B FRAM device.

12.1.1. Library

/SPI

12.1.2. Description

This block reads stored data from a F25L16B FRAM device. The data is read byte wise, so every time the block is executed one byte can be loaded of the FRAM. The storage address from which the byte is read is provided via the block’s input port.

12.1.3. Data Type Support

The block’s output data type is uint8.

The address input port expects a uint16 value.

12.1.4. Parameters and Dialog Box

12.1.4.1. SPI Port

Selects the SPI peripheral wich is configured by an SPI Master Config block.

12.1.4.2. Chip Select Port and Pin

Configuration of the MCU Pin the device’s chip select input is connected to.
