12.3. FM24CL16B FRAM Read

This block reads one byte stored by the FM24CL16B FRAM device.

12.3.1. Library

/I2C

12.3.2. Description

This block reads one single byte stored by the FM24CL16B FRAM device from the given address each time it is executed. The address range of the device is from 0 to 2047 which will be provided at the address input port. The address is partitioned internally into page and word address but the block address input expects one single address value inside the 2KB memory range. Since the page address is part of the I2C device address no device address must be configured in the parameter mask.

12.3.3. Data Type Support

The address input data type is uint16. The data output type is uint8. The error output type is boolean.

12.3.4. Parameters and Dialog Box

12.3.4.1. I2C Port

Selects the identifier of the I2C peripheral which is configured by an I2C Config block.
