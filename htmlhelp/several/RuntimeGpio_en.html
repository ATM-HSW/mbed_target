8.1. GPIO Runtime

This block toggels a GPIO pin for purpose of runtime measurements.

8.1.1. Library

utilities

8.1.2. Description

This block generates square wave signal at a GPIO pin to measure computational time of a model or subsystem. Therefor the block sets an output pin to high value at the very beging of the susystem’s/model’s code an resets the pin to low at the very end. So the duration of the high pulse is a measurement of the time needed to execute the code of the system the block is placed in.

8.1.3. Data Type Support

This block has no input or output ports.

8.1.4. Parameters and Dialog Box

8.1.4.1. Port Name

Select the port name of the GPIO pin to use.

8.1.4.2. Pin Number

Select the pin number of the GPIO pin to use.
