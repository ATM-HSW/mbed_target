10.2. PCF8574 Digital Output

Sets or clears the specified pin or sets the whole port of a PCF8574 port expander.

10.2.1. Library

/I2C

10.2.2. Description

This block interfaces the PCF8574 port expander chip. It enables setting a single pin of the PCF8574 or setting the whole port at once to a 8 bit pattern.

10.2.3. Data Type Support

The input port data type is uint8. The error output port data type is boolean.

10.2.4. Parameters and Dialog Box

10.2.4.1. I2C Port

Selects the identifier of the I2C peripheral which is configured by an I2C Config block.

10.2.4.2. Device Address

Bus address of the used PCF8574 device.

10.2.4.3. Output

Selects the single output pin of the PCF8574 to be written or the whole 8 bit port.

10.2.4.4. Enable error output port

Enables an error output port with return values:

  • “0” if no error occurred
  • “1” if there was an error during communication
