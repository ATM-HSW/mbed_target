PowerSTEP01_ReleaseSw
=====================================================

Description
-----------

This block issues a PowerStep01 Release SW command to the selected motor driver.
After the motor driver recived the PowerStep01 Release SW command a motion in the specified direction and with the specified minimum speed is performed, until the driver recognizes an rising edge on the switch input pin and the SW_F flag in the status register is forced low.
Then depending on the setting in the mask the specified action is performed and a Hard Stop command is issues.

   :width: 250 px

Parameters and Dialog Box
-------------------------

.. image:: ../images/powerSTEP01_ReleaseSw_Mask.png

SPI Port
.........
specify the SPI port number that should be used by the block.

Motor Number
............
specifies the motor in the daisychain.

SSEL Port/Pin
....................
specify the mcu pin which is connected to the chip select pin from the driver. This block supports the naming schema: 'P' + letter 'A', 'B', 'C', ... + '_' + number '0', '1', ... '15', e.g. PA_0.

Sample Time
...........
specifies the time interval between samples. To inherit the sample time, set this parameter to -1.

Action
......
specifies the action that should performed if the switch flag in the status register is forced low (switch input not connected to ground).
The possible actions correspond to the values defined in the POWERSTEP01 class: 

	ACTION_RESET --> ABS_POS register is reset
	
	ACTION_COPY --> ABS_POS register value is copied into the MARK register
