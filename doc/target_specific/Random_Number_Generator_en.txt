Random Number Generator
=======================

This block generates a random 32 bit integer value.


Library
-------

Targets/STM/DISCO_F407VG

Description
-----------

.. image:: ../images/Random_Number_Generator_block.png
   :scale: 40 %

This block generates random 32 bit integer values using the hardware random number generator (RNG) of an STM32 MCU. The generated numbers are true random numbers generated 
by an analog noise source. For more information about the generation circuit use the reference manual of the used STM32 MCU.

Data Type Support
-----------------

The block outputs an unsigned 32 bit integer (uint32).

Parameters and Dialog Box
-------------------------

This block has no parameters to be configured.
